Here are some of the projects I've worked on in my free time.

North American Happiness

This is a project that I built in collaboration with my roommates, Shreyas Chand and Brian Hou, for our CS 164 final project. This library allows multiple users on different machines to collaboratively make music together through a network connection. It was designed with programmers in mind, and very little musical knowledge is necessary to successfully compose complex and interesting musical pieces with other users. Fun fact: North American Happiness is also the name of the electronic rock band that we hope to start one day.

UC Berkeley UPE Website

This is the new website for the UC Berkeley Nu chapter of Upsilon Pi Epsilon, the national Computer Science honor society. As one of the former IT and web development chairs, I led a committee in designing and building the backend of this website using Django. The code for this project can be found on Github.

YouTube Adventure

This is a web app written in HTML / CSS / JS that allows the user to watch an endless stream of YouTube videos picked from an initial starting video. Users have the option to skip videos, pick how related they'd like the next video to be, and rewatch previous videos. I built this in collaboration with Shreyas Chand and Brian Hou, and the code for this project can be found on Github. Note that, due to an API issue, the site currently does not work.

Scheduler

This is a small program that I initially wrote for CS 61A to assign TAs to sections based on their availability and preferences. I later adapted the code for use by The Music Connection, to assign tutors to sites across the Berkeley Unified School District. The program uses lp_solve, a mixed-integer linear programming solver, to discover optimal assignments, and was designed such that a non-programmer could be easily taught how to use it.
