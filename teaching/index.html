teaching

For the past three semesters (Spring 2014, Fall 2014, and Spring 2015), I was a UGSI for CS 61A, the introductory computer science course. projects

Yelp Maps

This is a project that I developed along with fellow UGSI Brian Hou and Professor John DeNero. In this project, students construct a visualization of restaurant ratings for restaurants in the Berkeley area utilizing the Yelp Academic Dataset, using both existing user ratings as well as predicting ratings by analyzing user preferences. This project teaches concepts such as data abstraction, lists, and dictionaries, and also serves as an introduction into the field of machine learning.

More resources for CS 61A can be found on their course website.
